About images¶

Supported formats¶

png, dds, and tga are supported.

However, whether those files can be used with each plug-in depends on the plug-in. png can be used in most environments.

Mipmap¶

If _NoMip is at the end of a file name, it doesn't generate a mipmap. For example, Image.png and Image_NoMip.png results in the following image.
