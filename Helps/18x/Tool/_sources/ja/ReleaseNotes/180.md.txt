# 1.80 新機能

## 共通

### 衝突

パーティクルに衝突設定を追加しました。
地面や外部から与えた衝突形状に対して衝突判定を行い、バウンス、摩擦、寿命の減衰、座標系の指定を組み合わせて挙動を調整できます。

![](../../img/Reference/Collisions/collisions.gif)

### 軌跡、リボン

軌跡とリボンに、指定した長さごとにUVを繰り返す設定を追加しました。

### 回転

パーティクルの向きを移動方向に合わせて回転できるようになりました。

### トリガー/生成

生成設定を拡張し、`連続`と`トリガー`の2つの生成モードに対応しました。
あわせて、親消滅時・親衝突時のトリガーや、1回のトリガーで生成する数も設定できるようになりました。

### 外部モデル

生成元として使用する外部モデルを登録できるようになりました。
生成時と描画時の両方で、参照する外部モデルのインデックスと座標系（ローカル/ワールド）を指定できます。

### 描画

描画設定を拡張し、移動方向ビルボードを追加しました。
あわせて、UV左右反転確率、ミラーラップ、タイルの長さ、タイル（パーティクル毎）にも対応しました。

### モデル

モデルコンバータと対応フォーマットを更新し、`gltf / glb / obj / geo / bgeo` を直接読み込めるようになりました。

### マテリアル

マテリアルエディタに以下のノードを追加しました。
- スムースステップ
- 分岐 (Branch)
- 比較 (Compare)
- 論理積 (AND) / 論理和 (OR) / 論理否定 (NOT)
- `IsFrontFace`
- パーティクル時間
- ホワイトノイズ / セルラーノイズ
- RGBからHSV / HSVからRGB

### GPUパーティクル

GPU上でパーティクルを計算する機能を追加しました。
大量のパーティクルを効率よく扱えます。
※現時点ではプレビュー機能のため、環境によっては利用できない場合があります。

![](../../img/Reference/GpuParticles/gpu_particles.png)

### プロファイラ

CPU負荷とGPU負荷を計測して表示するプロファイラを追加しました。
※現時点ではプレビュー機能のため、環境によっては利用できない場合があります。

![Overview](../../img/Reference/Profiler/Profiler_Overview_Ja.png)

## ツール

### ビューア

ビューアの地面に対する衝突判定を有効にできるようになりました。

## Unity

- EffekseerForUnityを1.8系へ更新しました。
- `.efk`ファイルと依存リソースをまとめて取り込めるインポートメニューを追加しました。
- `EffekseerHandle`と`EffekseerEmitter`からトリガーを送信するAPIを追加しました。
- URP/HDRPのカスタムレンダーパスでレイヤーマスクを指定できるようになりました。
- Unity 6のURP RenderGraphに対応しました。
- WebGLプラグインは、使用するUnityのバージョンに応じて以下のものを利用します。
  - Unity 2021.2以降: `2.0.19`
  - Unity 2022.2以降: `3.1.8`
  - Unity 2023.2以降: `3.1.38`

## 不具合修正

### 共通

- GPUパーティクルの描画が正しく行われない不具合を修正しました。

### Unity

- Unity 6.3 の URP / RenderGraph 環境で `RenderGraphExecutionError` が発生する不具合を修正しました。
- `EffekseerEmitter` からの初回再生時に、回転、拡大率、表示状態、再生速度が正しく反映されない不具合を修正しました。
- RenderGraph と Vulkan を使用する Android 環境で、エフェクトが描画されない不具合を修正しました。
