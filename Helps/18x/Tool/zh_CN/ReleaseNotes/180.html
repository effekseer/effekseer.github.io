1.80 新功能

通用

碰撞

为粒子新增了碰撞设置。 可以将地面碰撞或外部碰撞与反弹、摩擦、寿命衰减和坐标系设置组合使用，以调整最终行为。

轨迹、丝带

为轨迹和丝带新增了按指定长度重复 UV 的设置。

旋转

粒子现在可以根据移动方向自动调整朝向。

触发器/生成

扩展了生成设置，新增 连续 与 触发器 两种生成模式。 同时支持父节点消失/父节点碰撞触发器，以及每次触发生成数量的设置。

外部模型

现在可以将外部模型注册为生成源。 在生成和渲染时，都可以指定外部模型的索引与坐标系（本地/世界）。

渲染

扩展了渲染设置，新增移动方向 Billboard。 同时支持 UV 水平翻转概率、镜像包裹、瓦片长度，以及按粒子平铺。

模型

更新了模型转换器与支持格式，现在可以直接加载 gltf / glb / obj / geo / bgeo。

材质

材质编辑器新增以下节点：

  • 平滑步进

  • 分支

  • 比较

  • 逻辑与（AND）/ 逻辑或（OR）/ 逻辑非（NOT）

  • 正面判定（IsFrontFace）

  • 粒子时间

  • 白噪声 / 蜂窝噪声

  • RGB转HSV / HSV转RGB

GPU 粒子

新增基于 GPU 的粒子计算功能。 可以更高效地处理大量粒子。 注意：该功能当前为预览版，在部分环境中可能不可用。

性能分析器

新增用于测量并显示 CPU/GPU 负载的性能分析器。 注意：该功能当前为预览版，在部分环境中可能不可用。

工具

预览器

现在可以为预览器中的地面启用碰撞判定。

Unity

  • 将 EffekseerForUnity 更新至 1.8 系列。

  • 新增用于导入 .efk 文件及其依赖资源的菜单。

  • 为 EffekseerHandle 与 EffekseerEmitter 添加了触发器发送 API。

  • URP/HDRP 自定义渲染通道支持图层遮罩。

  • 新增 Unity 6 的 URP RenderGraph 支持。

  • WebGL 插件会根据 Unity 版本自动选择：

    • Unity 2021.2 及以上：2.0.19

    • Unity 2022.2 及以上：3.1.8

    • Unity 2023.2 及以上：3.1.38

问题修复

通用

  • 修复了 GPU 粒子无法正确渲染的问题。

Unity

  • 修复了在 Unity 6.3 的 URP / RenderGraph 环境下发生 RenderGraphExecutionError 的问题。

  • 修复了从 EffekseerEmitter 首次播放时，旋转、缩放、显示状态与播放速度未被正确应用的问题。

  • 修复了在 Android 环境中使用 RenderGraph 与 Vulkan 时，特效无法渲染的问题。
