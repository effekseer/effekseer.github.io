APIリファレンス¶

EffekseerEmitter¶

継承: Spatial < Node < Object

Effekseerエフェクトの3D放出源。

説明¶

Effekseerのエフェクトを再生、描画するための3Dオブジェクトです。EffekseerEffectをセットし、play()することで再生することができます。また、設定されたトランスフォーム（位置、回転、拡大縮小）はエフェクトの描画に反映されます。

  • false: 自動再生を行わない

一時停止の設定。

  • true: 一時停止する

  • false: 一時停止しない

Getter get_speed()

再生速度の設定。範囲:0.0 ～ 10.0
