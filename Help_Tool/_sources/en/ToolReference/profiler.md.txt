# Profiler

## Overview
The Profiler is a tool for checking the load of effects being played in Effekseer.
It visualizes total CPU and GPU load for the system, as well as playback counts and GPU load for each effect.

![Overview](../../img/Reference/Profiler/Profiler_Overview_En.png)

## How to use

### Editor tab

Displays CPU and GPU load for effects running in the Editor in real time as line graphs.

- Press the `Clear Data` button to clear the recorded data.

### Target tab

![Target Profiler](../../img/Reference/Profiler/Profiler_Target_En.png)

Displays CPU and GPU load for effects running in games or applications, as well as playback counts and GPU load for each effect, in real time as line graphs.

Use this function as follows:

1. Configure the [Network](network.md) panel and connect to the target.
2. Return to the Profiler panel and confirm that the status is `Connected`.
3. Press the `Start Profile` button.
